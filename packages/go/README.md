# Swimburger - Go Module

Personal intro card for the terminal.

## Usage

Install:
```bash
go install github.com/swimburger/intro/packages/go/swimburger@latest
swimburger
```

Or run directly:
```bash
go run github.com/swimburger/intro/packages/go/swimburger@latest
```

## Local Development

```bash
go mod download
go run main.go
```

## Building

```bash
go build -o swimburger
./swimburger
```
